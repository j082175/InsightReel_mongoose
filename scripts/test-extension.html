🔧 Chrome 확장 기능 테스트

📊 클러스터 API 테스트

서버의 클러스터 수집 기능을 테스트합니다.

🎯 확장 프로그램 정보

현재 로드된 확장 프로그램 정보를 확인합니다.

📋 사용법

  1. Chrome 확장 로드: Chrome > 확장 프로그램 > 개발자 모드 ON > "압축해제된 확장 프로그램을 로드합니다" 클릭
  2. 확장 폴더 선택: extension 폴더 선택
  3. YouTube 채널 이동: 아무 YouTube 채널 페이지로 이동
  4. 버튼 확인: 구독 버튼 옆에 "📊 채널 수집" 버튼이 나타나는지 확인
  5. 기능 테스트: 버튼 클릭 후 키워드 입력 모달 확인
