Chrome Extension 동작 테스트

📋 테스트 단계

  1. Chrome에서 chrome://extensions/ 열기
  2. 개발자 모드 활성화
  3. 압축해제된 확장 프로그램을 로드합니다 클릭
  4. C:\Users\j0821\Documents\___JUNSOOCHO___\InsightReel\extension 폴더 선택

✅ 성공 확인 방법

  • Extension이 에러 없이 로드됨
  • Instagram/TikTok/YouTube 페이지에서 개발자 도구 콘솔에서:
    • 🚀 Content Script 시작 - 디버깅용 메시지 표시
    • 📡 서버 URL: http://localhost:3000 표시
    • 🔧 개발 모드: true 표시

🔧 확인할 환경변수

🌐 테스트 페이지 링크

  • Instagram
  • TikTok
  • YouTube
