# InsightReel Component Library

**InsightReel**은 YouTube, Instagram, TikTok 비디오를 자동으로 수집하고 AI로 분석하는 도구입니다.
이 Storybook은 프로젝트에서 사용되는 모든 컴포넌트들의 문서와 예시를 제공합니다.

## 🎯 프로젝트 개요

### 주요 기능
- 📺 **다중 플랫폼 지원**: YouTube, Instagram, TikTok 비디오 수집
- 🤖 **AI 분석**: Google Gemini API를 활용한 자동 비디오 분석
- 📊 **트렌드 모니터링**: 채널별 트렌딩 비디오 추적
- 📝 **자동 문서화**: Google Sheets 연동으로 분석 결과 저장
- 🔍 **고급 검색**: 키워드 기반 비디오/채널 검색

### 기술 스택
- **Frontend**: React 18 + TypeScript + Vite + Tailwind CSS
- **Backend**: Node.js + Express + MongoDB Atlas
- **AI**: Google Gemini API
- **Testing**: Jest + React Testing Library
- **Quality**: ESLint + Prettier + Storybook

## 🧩 컴포넌트 아키텍처

### Feature-Sliced Design (FSD)
프로젝트는 FSD 아키텍처 패턴을 따릅니다:

```
src/
├── app/          # 앱 레이어 (라우팅, 프로바이더)
├── pages/        # 페이지 레이어 (각 페이지 컴포넌트)
├── features/     # 기능 레이어 (비즈니스 로직 모듈)
└── shared/       # 공유 레이어 (재사용 컴포넌트)
    ├── components/   # 공통 컴포넌트
    ├── ui/          # 기본 UI 컴포넌트
    ├── hooks/       # 커스텀 훅
    ├── utils/       # 유틸리티 함수
    └── types/       # TypeScript 타입 정의
```
