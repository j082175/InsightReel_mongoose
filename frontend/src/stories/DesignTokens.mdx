## 📏 간격 (Spacing)

### 기본 간격 시스템
```css
/* Tailwind CSS 간격 시스템 (rem 기준) */
0    = 0px
1    = 0.25rem  (4px)
2    = 0.5rem   (8px)
3    = 0.75rem  (12px)
4    = 1rem     (16px)  ← 기본 단위
5    = 1.25rem  (20px)
6    = 1.5rem   (24px)
8    = 2rem     (32px)
10   = 2.5rem   (40px)
12   = 3rem     (48px)
16   = 4rem     (64px)
20   = 5rem     (80px)
24   = 6rem     (96px)
```

### 사용 예시
- **컴포넌트 내부 패딩**: `p-4` (16px)
- **컴포넌트 간 간격**: `space-y-6` (24px)
- **섹션 간 간격**: `mb-8` (32px)
- **페이지 여백**: `p-6` 또는 `p-8`

## 🔤 타이포그래피

### 폰트 패밀리
```css
font-sans: Inter, ui-sans-serif, system-ui, sans-serif
font-mono: ui-monospace, SFMono-Regular, "SF Mono", Consolas, monospace
```

### 텍스트 크기
| 클래스 | 크기 | 용도 |
|--------|------|------|
| `text-xs` | 12px | 캡션, 메타 정보 |
| `text-sm` | 14px | 보조 텍스트, 라벨 |
| `text-base` | 16px | 기본 본문 텍스트 |
| `text-lg` | 18px | 중요한 텍스트 |
| `text-xl` | 20px | 소제목 |
| `text-2xl` | 24px | 섹션 제목 |
| `text-3xl` | 30px | 페이지 제목 |

### 폰트 가중치
| 클래스 | 가중치 | 용도 |
|--------|--------|------|
| `font-normal` | 400 | 기본 텍스트 |
| `font-medium` | 500 | 강조 텍스트 |
| `font-semibold` | 600 | 중요한 라벨 |
| `font-bold` | 700 | 제목, 헤딩 |

## 📐 레이아웃

### 컨테이너 크기
```css
/* 반응형 컨테이너 */
container: 100%
sm: 640px
md: 768px
lg: 1024px
xl: 1280px
2xl: 1536px
```

### 그리드 시스템
- **기본**: `grid-cols-1` (모바일)
- **태블릿**: `md:grid-cols-2`
- **데스크톱**: `lg:grid-cols-3` 또는 `lg:grid-cols-4`

### 플렉스 레이아웃
- **가로 정렬**: `flex justify-center items-center`
- **세로 스택**: `flex flex-col space-y-4`
- **양쪽 정렬**: `flex justify-between items-center`

## 🎭 그림자 (Shadow)

### 그림자 시스템
| 클래스 | 효과 | 용도 |
|--------|------|------|
| `shadow-sm` | 미세한 그림자 | 카드 hover |
| `shadow` | 기본 그림자 | 기본 카드 |
| `shadow-md` | 중간 그림자 | 부유 카드 |
| `shadow-lg` | 큰 그림자 | 모달, 팝오버 |
| `shadow-xl` | 매우 큰 그림자 | 드래그 앤 드롭 |

## 🎯 상호작용 상태

### 호버 효과
```css
/* 기본 호버 */
hover:bg-gray-50
hover:shadow-md
hover:scale-105

/* 버튼 호버 */
hover:bg-blue-600 (Primary)
hover:bg-gray-600 (Secondary)
```

### 포커스 상태

```css
focus:ring-blue-500
focus:ring-offset-2
focus:outline-none
```

### 활성 상태
```css
/* 클릭/활성 상태 */

```

## 🎨 애니메이션

### 전환 효과
```css
/* 기본 전환 */
transition-all duration-200 ease-in-out

/* 느린 전환 */
transition-all duration-300 ease-in-out

/* 빠른 전환 */
transition-all duration-150 ease-in-out
```

### Framer Motion 설정
```javascript
// 기본 애니메이션
const fadeIn = {
  initial: { opacity: 0, y: 20 },
  animate: { opacity: 1, y: 0 },
  transition: { duration: 0.2 }
}

// 스케일 애니메이션
const scaleIn = {

```


## ♿ 접근성 가이드라인

### 색상 대비
- **텍스트**: 최소 4.5:1 비율
- **대형 텍스트**: 최소 3:1 비율
- **비텍스트**: 최소 3:1 비율

### 포커스 관리
- 모든 인터랙티브 요소에 명확한 포커스 표시
- 키보드 네비게이션 지원
- 논리적인 탭 순서

### ARIA 라벨링
- 의미 있는 텍스트 라벨
- 상태 정보 전달
- 관계성 명시
